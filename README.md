## ◈ Running Locally

```bash
# Clone it
git clone https://github.com/yourusername/portfolio.git
cd portfolio

# Install
npm install

# Fire it up
npm run dev
# → http://localhost:3000
```

No `.env` required for local dev. Optionally add analytics keys if you want tracking.
